# Pattern Generator and Number Analyzer

A simple Python console application that demonstrates the use of **loops**, **conditional statements**, and **control flow** by generating star patterns and analyzing numbers within a given range.

## Features

- ⭐ Generate a right-angled triangle star pattern
- 🔢 Analyze numbers in a given range
- ✅ Identify even and odd numbers
- ➕ Calculate the sum of all numbers in a range
- 🔁 Interactive menu-driven program
- 🎯 Demonstrates Python control statements (`if`, `else`, `for`, `while`, `break`, `continue`, and `pass`)

## Technologies Used

- Python 3

## Program Flow

1. Display a welcome message.
2. Generate a right-angled triangle pattern.
3. Analyze numbers in a specified range.
4. Demonstrate the use of control statements.
5. Provide a menu with the following options:
   - Generate Triangle Pattern
   - Number Analysis
   - Exit
6. Display a thank-you message before exiting.

3. Open a terminal or command prompt.
4. Navigate to the project folder.
5. Run the following command:

```bash
python pattern_generator.py
```
